# โจทย์: DOLE Book Store — ระบบจัดการสื่อการเรียนรู้

> โน้ตต้นทาง (source brief) สำหรับให้เคาะดีญะฮ์ประเมิน — สรุปจากเอกสารถ่ายรูป 3 หน้า
> ลูกค้า: กรมส่งเสริมการเรียนรู้ (กสร.) · Department of Learning Encouragement

## ภาพรวม
แพลตฟอร์มกลาง "DOLE Book Store / ระบบจัดการสื่อการเรียนรู้" — คลังหนังสืออิเล็กทรอนิกส์
(e-book PDF) + ระบบสั่งซื้อ/เบิกสื่อสำหรับโรงเรียน ต่อยอดเชื่อม SSO, Big Data,
และ AI Adaptive Personalized Learning

## เอกสารต้นทาง (3 หน้า)
- **7.1.3 ค่าใช้จ่ายรายฟังก์ชัน** — 5 โมดูลหลัก:
  1. ระบบจัดการสิทธิ์ผู้ใช้ (ผู้เรียน/ครู/ผู้บริหารโรงเรียน/ผู้ดูแลระบบ)
  2. ระบบจัดการหนังสืออิเล็กทรอนิกส์ — อัปโหลด PDF, จัดหมวดตามวิชา/ระดับชั้น/ประเภท, API เชื่อมโรงพิมพ์
  3. ระบบจัดการคำสั่งซื้อ — เลือกหนังสือจากคลัง
  4. ระบบประวัติการสั่งซื้อ
  5. ระบบรายงานและสถิติ — ยอดดาวน์โหลด, คำสั่งซื้อจำแนกรายจังหวัด
- **7.1.1 Diagram** — ecosystem: SSO, DOLE Learner Data, e-Market, Digital Wallet/EKAS/Credit Bank/
  e-Portfolio, Big Data, AI Adaptive Learning (Claude/Gemini/OpenAI), เชื่อม HRMS/E-Saraban (เฟส 1/2)
- **7.1.2 แนวคิดการออกแบบ** — ยืนยันชื่อระบบหลัก DOLE Book Store + Single Sign-On

## ขอบเขตที่ตกลงประเมิน
รวมทั้ง ecosystem (Book Store + SSO + Big Data + AI)
